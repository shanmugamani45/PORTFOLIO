Frequently Asked Questions

Do you take freelance projects?

+

Yes, depending on my current capacity. Please reach out via the contact section so we can discuss the scope and timeline.

What technologies do you specialize in?

+

I primarily build highly interactive user interfaces using React, Next.js, and Three.js (WebGL). I also use Framer Motion and GSAP for micro-interactions.

What is your approach to AI integration?

+

I believe AI should enhance user experience, not complicate it. I focus on integrating machine learning models seamlessly into the backend while exposing them through clean, intuitive frontend interfaces.

How long does a typical project take?

+

Project timelines vary based on complexity. A standard interactive web application might take 4-6 weeks, while complex AI-integrated platforms can take several months. We establish clear milestones before starting.

Do you work with existing design systems?

+

Yes, I am comfortable adapting to established brand guidelines, but I also specialize in building custom, highly cinematic design systems from the ground up tailored to specific project needs.

What do you need from me to start a project?

+

Typically, a clear project brief, timeline expectations, and any existing brand assets or data requirements. We kick off every project with a detailed discovery phase to align on goals and deliverables.
